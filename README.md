## Using poetry 

#### Starting new project in poetry
```
	_poetry new {project_name}_
```
Go to project directory
```
	cd {project_name}
```
Install vertual env
```
	poetry install
```
Activate venv
```
	.venv\Scripts\activate
```
Intall required packages
```
	poetry add {package_name}
```
Running scripts
```
	poetry run python {file_name.py}
```
